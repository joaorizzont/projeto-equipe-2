# Model Instructions
NÃO invente APIs que não foram explicitamente solicitadas. Se uma funcionalidade não existe, você DEVE consultar o arquivo .agents/rules/backend.md para ver se a API foi definida ou se deve ser criada.
Sempre que for criar uma API:
1. Verifique se ela já existe em .agents/rules/backend.md
2. Se não existir, consulte .agents/rules/backend.md para entender as convenções de nomenclatura e estrutura
3. Crie a API seguindo estritamente as convenções definidas no arquivo
4. Se não tiver certeza, pergunte ao usuário antes de prosseguir
